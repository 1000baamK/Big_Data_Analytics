#### 2. 데이터 분석
```
수집된 데이터를 기반으로 데이터를 분석하는 단계!
분석을 통해 데이터의 특성, 패턴을 파악하여 인사이트를 도출하는 과정이고 이 단계에서는 기술 통계량, 머신러닝, 딥러닝
등 다양한 분석 기법을 사용하여 데이터의 숨겨진 의미를 발견한다.
```
- T-Test(T-검정)

#### 3. 시각화
```
데이터 분석결과를 시각적으로 표현하는 단계!
```
